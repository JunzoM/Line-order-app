# 📦 LINE発注システム v2.0（完全版）

商品登録・発注履歴管理・複数ユーザー対応の発注管理システムです。

## ✨ 主な機能

### 🔐 ユーザー管理
- ログイン/ログアウト機能
- 権限管理（管理者/一般ユーザー）
- ユーザー登録画面（管理者のみ）

### 📦 商品管理（管理者のみ）
- 商品の登録・編集・削除
- カスタム単位設定
- カテゴリ・仕入先・単価管理

### 📝 発注機能
- 登録済み商品から選択
- 数量・備考入力
- 確認画面表示

### 📱 LINE通知
- 美しいカードデザイン
- 発注日時表示
- 商品名と数量（数量は赤字で強調）
- 発注者名表示

### 📊 発注履歴
- 全発注の記録・閲覧
- ステータス管理
- フィルタリング機能
- 統計ダッシュボード

### 🌐 PWA対応
- スマホのホーム画面に追加可能
- アプリのような使い心地
