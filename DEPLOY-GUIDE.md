## 🚀 クイックスタート（Render推奨）

### 必要なもの
- GitHubアカウント
- 5分の時間

### 手順
1. **GitHubにコードをアップロード**
```bash
git init
git add .
git commit -m "Initial commit"
# GitHubでリポジトリ作成後
git remote add origin https://github.com/your-username/line-order-app.git
git push -u origin main
```

2. **Render.comにログイン**
   - https://render.com/
   - GitHubで連携

3. **新しいWeb Serviceを作成**
   - リポジトリを選択
   - Freeプランを選択
   - Build Command: `npm install`
   - Start Command: `npm start`

4. **環境変数を設定**
   ```
   LINE_CHANNEL_ACCESS_TOKEN = あなたのトークン
   LINE_USER_ID = あなたのユーザーID
   ```

5. **デプロイ完了！**
   - URLが自動生成されます
   - 例: `https://line-order-app.onrender.com`

詳細は **DEPLOY-RENDER.md** を参照！

## 💡 スリープ対策（Renderの場合）

1. https://uptimerobot.com/ でアカウント作成

2. モニターを追加
   - URL: `https://あなたのアプリ.onrender.com/api/health`
   - Interval: 5分

3. 完了！これで常時起動状態を維持
